Stone Workshop - Taller de Piedra Natural ========================================== Módulo especializado para talleres de piedra natural. Procesos soportados: - **Acabado**: Entrada de placa completa → misma placa con diferente acabado (lote + sufijo) - **Corte**: Entrada de placa completa → formatos/piezas más pequeñas (lote + sufijo) - **Otro**: Procesos personalizados Características: - Panel visual con carrito persistente - Cálculo automático de costos por m² - Trazabilidad de lotes con sufijo de proceso - Integración con MRP
